# Customer & Revenue Segmentation Analysis using Python

## Project Overview
This project analyzes retail sales data to identify high-value customer segments, top-performing product categories, and geographic revenue concentration.

## Objectives
- Analyze transaction distribution by gender and age group
- Identify top-performing states and professions
- Evaluate revenue contribution and average order value
- Provide data-driven business recommendations

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
